## About Project
### Requirements
- [X] fetch 통신을 이용하여 서버에서 데이터 받아와서 뿌리기
  - [X] ajax 통신으로 구현된 코드들을 fetch 통신으로 변경.
- [X] 검색기능 구현
- [X] 페이징 기능 구현
  - [X] 검색된 데이터 수에 알맞게 페이지 수 조정.
  - [X] 페이지 번호를 누르면 해당 페이지로 이동 
  - [X] 이전을 누르면 처음으로, 다음을 누르면 마지막으로 이동.
    - [ ] 이전, 다음 버튼과 처음, 마지막 버튼을 구별하여 생성
